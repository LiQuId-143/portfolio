# LiQuId — Portfolio

Fully self-hosted copy of Romanenkov Kirill's portfolio. Renders identically
to the source site including all animations and interactions, with zero
Framer dependencies or branding.

## Structure

```
/
├── index.html, about.html, projects.html
├── projects/               # case studies
├── sites/                  # JS runtime modules (preserved paths)
├── modules/                # Framer custom components
├── assets/                 # woff2 fonts
├── images/                 # png / jpg / gif
└── cms/                    # collection data
```

## Run locally

```
python3 -m http.server 8000
```

Open <http://localhost:8000/>.

## Deploy

Push to GitHub Pages, Netlify, or Vercel. No build step.

## Generated by

A Playwright pipeline that renders every page in headless Chromium,
captures every network response, preserves the original URL paths so the
runtime's internal ES-module imports keep working, then rewrites
`framerusercontent.com` URLs to local relative paths in HTML / JS / CSS.

Framer-specific artifacts removed:

- "Made in Framer" badge
- On-page editor scripts
- `events.framer.com` analytics
- Dynamic imports to `framer.com/edit/*` and `framer.com/m/*`
